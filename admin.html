Configuração do Servidor de Música

Para transmitir músicas na sua web rádio, você precisa de um servidor de streaming. Abaixo estão as opções mais populares com instruções passo a passo.

Se configurado, o player usará este stream ao vivo em vez de arquivos locais

Opções de Servidores Populares

Icecast2 (Gratuito / Open Source)

O servidor de streaming mais usado para web rádios. Gratuito, código aberto e muito confiável.

Gratuito (necessita VPS/servidor próprio)

  • Suporte a MP3 e OGG Vorbis
  • Múltiplos pontos de montagem
  • Painel web de administração
  • Estatísticas de ouvintes em tempo real
  • Compatível com Linux, Windows e macOS

Instalação no Ubuntu/Debian

  1. Contrate um VPS (DigitalOcean, Vultr, Linode, Contabo, etc.) com Ubuntu 22.04+
  2. Acesse o servidor via SSH: ssh root@seu-ip
  3. Atualize o sistema: sudo apt update && sudo apt upgrade -y
  4. Instale o Icecast2: sudo apt install icecast2 -y
  5. Configure durante a instalação: defina hostname, senhas de admin e source
  6. Edite o arquivo de configuração: sudo nano /etc/icecast2/icecast.xml
  7. Ajuste as senhas, porta (padrão 8000) e limites de ouvintes
  8. Inicie o servidor: sudo systemctl start icecast2
  9. Habilite inicialização automática: sudo systemctl enable icecast2
  10. Acesse o painel: http://seu-ip:8000
  11. Instale o Liquidsoap para automatizar playlists: sudo apt install liquidsoap -y
  12. Crie o script de playlist em /etc/liquidsoap/radio.liq
  13. Coloque a URL do stream no campo acima: http://seu-ip:8000/stream

SHOUTcast (Freemium)

Plataforma clássica e confiável para streaming de áudio. Versão DNAS gratuita disponível.

Gratuito (DNAS) / Planos pagos a partir de $4.95/mês

  • Suporte a MP3 e AAC+
  • Painel de controle web
  • Suporte a playlists automáticas
  • Autosave e recuperação
  • Listagem no diretório SHOUTcast

Instalação

  1. Acesse shoutcast.com e baixe o SHOUTcast DNAS
  2. Contrate um VPS ou servidor dedicado
  3. Faça upload do DNAS para o servidor
  4. Extraia: tar -xvf sc_serv*.tar.gz
  5. Configure o arquivo sc_serv.conf: defina porta, senha e limites
  6. Inicie: ./sc_serv sc_serv.conf
  7. Use o SHOUTcast Transcoder ou BUTT para enviar o áudio
  8. Acesse: http://seu-ip:8000
  9. Coloque a URL do stream no campo acima

AzuraCast (Gratuito / All-in-One)

Solução completa e moderna para web rádios. Instalação com Docker, painel completo e fácil de usar.

100% Gratuito e Open Source

  • Painel de administração completo e moderno
  • Upload de músicas via navegador
  • Playlists automáticas e agendamento
  • Estatísticas detalhadas
  • Suporte a Icecast e SHOUTcast integrados
  • Instalação com um comando
  • Recomendado para iniciantes!

Instalação com Docker (Recomendado)

  1. Contrate um VPS com Ubuntu 22.04+ (mínimo 2GB RAM)
  2. Acesse via SSH: ssh root@seu-ip
  3. Instale com um comando: curl -fsSL https://raw.githubusercontent.com/AzuraCast/AzuraCast/main/docker.sh | bash
  4. Siga as instruções na tela para configurar
  5. Acesse o painel: http://seu-ip
  6. Crie sua primeira estação de rádio pelo painel
  7. Faça upload das músicas diretamente pelo navegador
  8. Crie playlists e configure horários
  9. Copie a URL do stream público e cole no campo acima
  10. Pronto! Sua rádio está no ar! 🎉

Serviços de Hospedagem (Pronto para Usar)

Se você não quer configurar servidor, contrate um serviço pronto. Apenas envie suas músicas e comece a transmitir.

A partir de R$29,90/mês

  • Radiol.com.br — Planos a partir de R$29,90/mês
  • HostMídia — Streaming profissional
  • Maxcast — Servidor para web rádios
  • StreamBR — Hospedagem nacional
  • Zeno.fm — Plano gratuito disponível (com anúncios)

Como Contratar e Configurar

  1. Escolha um dos serviços acima e crie uma conta
  2. Selecione o plano adequado (quantidade de ouvintes simultâneos)
  3. Após a ativação, você receberá os dados do servidor (IP, porta, senha)
  4. Faça upload das suas músicas pelo painel do serviço
  5. Configure playlists e horários de programação
  6. Copie a URL do stream fornecida pelo serviço
  7. Cole a URL no campo "URL do Stream de Áudio" acima
  8. Salve as configurações e teste no seu site!

BUTT (Broadcast Using This Tool)

Software gratuito para transmitir ao vivo do seu computador para o servidor Icecast ou SHOUTcast.

Gratuito

  • Transmita ao vivo do seu PC
  • Compatível com Icecast e SHOUTcast
  • Windows, macOS e Linux
  • Ideal para locutores ao vivo

Configuração do BUTT

  1. Baixe o BUTT em danielnoethen.de/butt
  2. Instale e abra o programa
  3. Clique em "Settings" → "Server"
  4. Adicione seu servidor: IP, porta, senha e ponto de montagem
  5. Configure a qualidade do áudio (128kbps MP3 recomendado)
  6. Selecione a entrada de áudio (microfone ou placa de som virtual)
  7. Clique em "Play" para iniciar a transmissão
  8. Para usar com Icecast: selecione tipo "Icecast" e preencha os dados
